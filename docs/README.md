# DBVC Documentation Library

This is the repo-level documentation entry point for DBVC. It is a router, not a requirement to read every doc.

## Start Here

- `agent-entrypoints.md` - choose the smallest useful docs path for a task.
- `roadmap.md` - active, proposed, completed, and archived work index.
- `requests.md` - unresolved documentation questions and requested corrections.

## Sections

- `architecture/` - long-lived explanations of DBVC systems.
- `development/` - setup, testing, QA, and local workflow docs.
- `implementation/` - active, proposed, and completed implementation guides.
- `reference/` - lookup docs for contracts, terms, usage, and fixtures.
- `troubleshooting/` - known problems and fix paths.
- `archives/` - historical or superseded docs that should not guide new work.
- `_meta/` - docs-library maintenance files.

Do not use archived docs as current guidance unless a current entry point links to them for historical context.
