# Suggested quick flow

1. Place `repomix-starter-kit/` in the target repo root.
2. Open the repo in Codex / Claude Code / Cursor.
3. Paste the contents of `INSTALL_PROMPT.md`.
4. Let the agent inspect the repo and adapt the templates.
5. Review the final repo-specific files.
6. Remove the starter kit folder if you no longer want it checked into the repo.

# Expected final files in the repo

- repomix.config.json
- .repomixignore
- repomix-instruction.md
- docs/ai/repomix-preflight.md (or equivalent)
- docs/ai/repomix-maintenance.md (or equivalent)
- a small appended section in AGENT.md / AGENTS.md / CODEX.md / CLAUDE.md
