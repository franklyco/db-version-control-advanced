## Repomix maintenance note

Before major implementation, refactors, architecture work, or handoff generation, quickly check whether the current Repomix config, ignore rules, instruction file, and maintenance note still match the repository structure.

If repo structure, directives, or other repo-context systems changed, update stale Repomix-related files first so packed context stays accurate and low-noise.

Keep Repomix aligned with existing authoritative repo docs, indexes, maps, graphs, and workflow guidance rather than duplicating them.
