# Package Layout

Status: Current
Last verified: 2026-06-22
Source of truth: `includes/Dbvc/AiPackage/SamplePackageBuilder.php`, `includes/Dbvc/AiPackage/CompactSchemaBuilder.php`, `includes/Dbvc/AiPackage/SubmissionPackageValidator.php`
Read when: distinguishing a DBVC-generated sample package from an AI-returned submission package.
Minimum context: `ai-agent-quickstart.md`.

## Fast Path

DBVC generates a `dbvc_ai_sample_package` for AI context. The AI returns a `dbvc_ai_submission_package` for DBVC intake.

Do not upload the sample package as the returned package. DBVC blocks sample packages during submission intake.

## Current Contract

Compact sample packages generated by DBVC contain:

```text
dbvc-ai-manifest.json
START_HERE.md
SCHEMA_COMPACT.json
samples/posts/{post_type}.json
samples/posts/{post_type}.context.json
samples/terms/{taxonomy}.json
samples/terms/{taxonomy}.context.json
```

Returned AI submission packages contain:

```text
dbvc-ai-manifest.json
entities/posts/{post_type}/{slug}.json
entities/terms/{taxonomy}/{slug}.json
```

Optional returned files may be included:

```text
docs/NOTES.md
reports/generation-summary.md
```

## Authoring Rules

- Put `dbvc-ai-manifest.json` at the ZIP root.
- Use `package_type: "dbvc_ai_submission_package"` in returned packages.
- Use `package_schema_version: 1`.
- Copy the sample package fingerprint into `source_sample_package.site_fingerprint`.
- Do not place returned entities under `samples/`.
- Do not copy the sample manifest as the returned manifest.

## Nuance

The DBVC sample package manifest stores its source site fingerprint at root-level `site_fingerprint`. The returned submission manifest stores that same value under `source_sample_package.site_fingerprint`.

DBVC currently accepts some legacy manifest mistakes with warnings, but current packages should use the canonical nested source sample package field.

Full reference sample packages may include richer split schema docs and sibling markdown files. The compact profile is the default and is the preferred browser-AI context shape.
